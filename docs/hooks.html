Kernel Integration Guide

NexusSU supports pre-GKI kernels (4.9, 4.14, 4.19, 5.4). You can integrate NexusSU into your kernel source using the Automated Script (Recommended) or by manually applying the diffs.

Method 1: Automated Script (Recommended)

The easiest way to integrate NexusSU is to use the provided apply.sh script located in the kernel/ directory of our GitHub repository.

$ chmod +x apply.sh
$ ./apply.sh /path/to/your/kernel/source

Note: If the script fails due to heavy ROM source modifications, proceed to Method 2.

Method 2: Manual Source Injection

Lines marked with + are mandatory additions. This includes credential escalation, VFS stealth, and the SELinux MAC bypass (maintaining global Enforcing mode).
