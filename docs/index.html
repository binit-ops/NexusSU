NexusSU

A kernel-native, zero-file root solution for Android.

Architecture: ARM64 Kernel 4.9 Kernel 4.14 Kernel 4.19 Kernel 5.4 

/* Zero-file privilege escalation via magic prctl */
#define PR_NEXUSSU 0x4E535553
#define CMD_ESCALATE 3

int main() {
    // Request Ring 0 Escalation
    prctl(PR_NEXUSSU, CMD_ESCALATE, 0, 0, 0);
    
    // Process is now running as global root
    execl("/system/bin/sh", "sh", NULL);
}

Syscall Hooking

Utilizes prctl magic numbers for kernel-to-userspace communication. No /dev/ nodes, zero filesystem traces.

Zero KABI Breakage

Tracks privileges via a lightweight PID hash map rather than modifying task_struct. Retains 100% compatibility with pre-compiled vendor drivers.

Dynamic SELinux Bypass

Dynamically grants target processes full MAC permissions while keeping the system globally in Enforcing mode.

Zero I/O Penalty Stealth

Actively filters root artifacts in /proc/mounts and VFS lookups using targeted sequence hooks, completely avoiding global I/O degradation.
