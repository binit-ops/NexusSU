Building the Engine

Because legacy GKI 1.0 and pre-GKI kernels do not support universal loadable modules, the NexusSU hooks must be compiled directly into the kernel source tree.

1. Prepare the Build Environment

Termux / PRoot 

apt update && apt upgrade -y
apt install git make bc bison build-essential zip curl flex \
    libssl-dev libfl-dev wget gcc-aarch64-linux-gnu -y

2. Fetch the Toolchain & Source

Repository Setup 

git clone --depth=1 https://github.com/kdrag0n/proton-clang.git ~/toolchain
git clone --depth=1 https://github.com/OEM/kernel_source.git ~/android-kernel
cd ~/android-kernel

3. Link the Makefiles

Makefile Injections 

echo "obj-y += nexussu_state.o" >> kernel/Makefile
echo "obj-y += nexussu.o" >> drivers/char/Makefile

4. Cross-Compile

Build Command 

export PATH="$HOME/toolchain/bin:$PATH"
export CROSS_COMPILE=aarch64-linux-gnu-
export CC=clang
export ARCH=arm64

make clean && make mrproper
make vendor/your_device_defconfig O=out
make -j8 O=out
Critical Warning

Always ensure you have a fallback method to restore your partitions. A single misaligned pointer in your source injection will cause an immediate bootloop.
