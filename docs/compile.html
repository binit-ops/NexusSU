Building the Engine

Because legacy GKI 1.0 kernels do not support universal loadable modules, the NexusSU hooks must be compiled directly into the kernel source tree.

1. Prepare Environment

Termux/Proot 

apt update && apt upgrade -y
apt install git make bc bison build-essential zip curl flex \
    libssl-dev libfl-dev wget gcc-aarch64-linux-gnu -y

2. Fetch & Link

Makefile Injections 

echo "obj-y += nexussu_state.o" >> kernel/Makefile
echo "obj-y += nexussu.o" >> drivers/char/Makefile

3. Build

Build Command 

# Clean the build directory
make clean && make mrproper

# Generate the config targeting ARM64
make ARCH=arm64 vendor/your_device_defconfig O=out

# Compile the kernel with the cross-compiler installed in Step 1
make -j$(nproc --all) \
    O=out \
    ARCH=arm64 \
    CROSS_COMPILE=aarch64-linux-gnu-
