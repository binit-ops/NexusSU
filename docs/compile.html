Compilation Guide

This guide covers how to compile both the NexusSU Manager App and the patched kernel.

Part 1: Building the Manager App (APK)

The Manager App is built using Gradle. The build process automatically uses the Android NDK to compile the native su binary and the nexussu_daemon from C source code, packaging them into the APK's assets folder.

Method A: GitHub Actions (Recommended)

The easiest way to build the Manager App is via GitHub Actions. The repository includes a workflow that handles the entire build process.

  1. Push your code to the main branch of your GitHub repository.
  2. Go to the Actions tab on GitHub.
  3. The workflow will automatically download Gradle 8.11.1, compile the native C code, and build the APK.
  4. Once complete, download the NexusSU-Manager-APK artifact from the workflow summary page.

Method B: Local Build (Android Studio)

If you prefer to build locally, ensure you have Android Studio and the Android NDK (version 25.1.8937393 or newer) installed.

$ git clone https://github.com/your-username/NexusSU.git
$ cd NexusSU
$ ./gradlew assembleRelease

The compiled APK will be located in manager/build/outputs/apk/release/.

Part 2: Building the Kernel

Once you have patched your kernel source using the apply.sh script (as described in the Kernel Integration guide), you can compile the kernel using the standard Android build process.

Terminal 💻 Kernel Build Environment 

# 1. Setup your environment variables
export ARCH=arm64
export SUBARCH=arm64
export CROSS_COMPILE=aarch64-linux-android-
export CROSS_COMPILE_COMPAT=arm-linux-androideabi-

# 2. Navigate to your patched kernel source
cd /path/to/your/kernel/source

# 3. Load your device's defconfig
make OEM_defconfig

# 4. Build the kernel
make -j$(nproc)

Once compilation is finished, the kernel image (usually Image.gz-dtb or Image) will be located in arch/arm64/boot/. You can then pack this into an AnyKernel3 zip or integrate it directly into your ROM's device tree.

Important Note on Boot Daemons

The NexusSU Manager App installs the nexussu_daemon to /data/adb/nexussu/bin/ on first launch. For the daemon to run on boot, ensure your ROM's init.nexussu.rc script is configured as detailed in the ROM Integration Guide.
