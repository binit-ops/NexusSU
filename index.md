# NexusSU
**NexusSU** is a powerful, kernel-based root solution for Android, designed to provide deep system control while remaining systemless and undetectable.

## Features
* **Kernel-Native:** Directly hooks into the Linux kernel for superior performance.
* **Systemless:** Keeps your `/system` partition untouched.
* **Unified Management:** One app to rule all your root permissions.
